# Contact

If you would like to get in touch with Open Water Lab (OWL) team, please use one of the following options.

## GitHub
For questions, bug reports, or feature requests related to specific tools, please open an issue in the corresponding repository.

- **[OpenWaterLab GitHub](https://github.com/OpenWaterLab)**

## Discussions

General questions about the ecosystem can be discussed in the GitHub Discussions section.

## Collaboration

If you are interested in collaborating, contributing tools, or integrating models into the OWL ecosystem, please contact the maintainers.

## Institutional affiliation

Open Water Lab (OWL) is part of the activities of the **[BIOMATH](https://biomath.ugent.be/) research group** at **[Ghent University](https://www.ugent.be/en), Belgium**.
